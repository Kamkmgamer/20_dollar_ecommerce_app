# Micro Business Store - $20 Budget

## Overview
- **Budget**: $20
- **Target Business**: Unserious / Side hustle
- **Max Products**: 1-10

## Tech Stack
- WordPress with Default Theme (Twenty Twenty-Five)
- Simulated with Next.js for demo purposes
- Static product data
- Contact-based ordering (no cart/checkout)

## Features
- Single product page display
- Simple "Contact to Order" approach
- WordPress default theme styling
- WhatsApp suggestion prominently displayed

## Implementation
- Company name: "Your Company Name"
- Prominent message: "Dropping a WhatsApp number would be better"
- WordPress-style footer: "Proudly powered by WordPress"
- Simple, basic styling similar to WordPress default themes
- No cart functionality - direct contact for orders

## Performance
- Naturally slower (PHP-based, server-side rendering)
- No optimization, basic CSS
- Expected: 2-4s page load time
